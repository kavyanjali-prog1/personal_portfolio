Job Portal Project in ServiceNow (Basic Level)

This project involved creating a basic job portal on the ServiceNow platform, providing a streamlined process for job seekers to view and apply for job openings.

Step-by-Step Guide:

  1. Create a new application:
    • Logged in to ServiceNow instance.
    • Navigated to System Applications > Create New Application.
    • Filled in application details: Name "Job Portal", Description "A portal for job seekers to apply for jobs".
  2. Create tables:
    • Created a "Job" table with fields: Job Title (string), Job Description (string), Job Requirements (string).
    • Created a "Job Application" table with fields: Job (reference to Job table), Applicant Name (string), Applicant Email (string), Resume (attachment).
  3. Create forms:
    • Created a form for the "Job" table with its fields.
    • Created a form for the "Job Application" table with its fields.
  4. Create a job portal page:
    • Navigated to Content Management > Pages.
    • Created a new page for the job portal, adding a title and description.
    • Added a Job List widget to display available jobs and a Job Application form for user submissions.
  5. Configure security:
    • Created roles for job seekers and hiring managers (User Administration > Roles) with relevant permissions.
    • Assigned these roles to users (User Administration > Users).
  6. Test and publish:
    • Tested the job portal for expected functionality.
    • Published the job portal page (Content Management > Pages).

This basic setup allows users to view job openings, apply for jobs, and submit their resume and application details.

Basic Functionality:

  • View available job openings.
  • Apply for jobs.
  • Submit their resume and application details.

Next Steps (Potential Enhancements):

  • Implement job search functionality.
  • Add application status tracking for job seekers.
  • Configure notifications for hiring managers.
  • Customize the portal's branding to match organizational guidelines.

Autonomous Landmine Detection

This project focuses on creating an autonomous mobile robotic platform for the detection and location tracking of landmines, significantly improving safety in demining operations. It integrates IoT capabilities with embedded sensor modules.

Components and functionalities:

  • ESP32 Microcontroller: Central processing unit for sensor data and communication.
  • Metal Detector: Identifies the presence of metallic objects (landmines).
  • GPS Module: Provides precise location coordinates of detected landmines.
  • BO Motors: Powers the mobile robotic platform for movement.
  • Buzzer System: Provides audible alerts upon detection.
  • IoT Protocols: Enables real-time alerts and live video streaming to a remote monitoring station.
  • Rechargeable Battery: Ensures extended operational time.

This project highlights my skills in IoT, embedded systems, and robotics, demonstrating an ability to develop practical solutions for critical applications.
